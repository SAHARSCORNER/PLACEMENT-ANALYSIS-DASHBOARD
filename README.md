# Placement Data Dashboard Dataset

## Overview

This project contains a placement dataset designed for analysis and visualization in Power BI, Excel, Tableau, or other Business Intelligence tools.

The dataset includes student placement records such as department, year of passing, enrollment number, PRN number, and employer details. It can be used for creating placement dashboards, tracking recruitment trends, and performing educational data analysis.

---

## Dataset Information

### File
- Dummy_Placements_Data.csv

### Records Included
- Student Information
- Academic Department Details
- Year of Passing
- Enrollment Numbers
- PRN Numbers
- Employer Information

---

## Dataset Fields

| Column Name | Description |
|------------|-------------|
| Sr.No | Unique serial number for each record |
| Student Name as per college record | Student's full name |
| Department | Academic department/program |
| YearofPassing | Graduation year |
| CollegeEnrollmentNumber | College enrollment identifier |
| PRNNumber | Permanent Registration Number |
| EmployerName | Company where the student was placed |
